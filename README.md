## CUri \- A URL parser class


</div>

### Description

This is a URL parser class that accepts a URL (such as "http://www.blah.com/") and parses out:<br>

- the protocol<br>

- the hostname<br>

- the username<br>

- the password<br>

- the port<br>

- the resource<br>

- the querystring<p />

Enjoy!
